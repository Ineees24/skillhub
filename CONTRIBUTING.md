# Contribution

Conservez les noms de dossiers existants : `skillhub-back` et `skillhub-front`.

Avant d'ouvrir une pull request :

1. Lancez les tests backend depuis `skillhub-back`.
2. Lancez le lint et le build frontend depuis `skillhub-front`.
3. Laissez le workflow GitHub Actions executer l'analyse Sonar.
